# 🐍 SnakeGame (WPF)

Красивая реализация классической игры **Snake** на WPF (.NET).

## 🚀 Возможности

* Классический геймплей змейки
* Подсчет очков
* Увеличение скорости с ростом змейки
* Game Over экран
* Перезапуск игры

---

## 🛠️ Технологии

* C#
* WPF (.NET)
* MVVM / Code-behind (если используется)

---

## 📦 Установка

1. Клонируй репозиторий:

```bash
git clone https://github.com/B18ack/SnakeGame-WPF.git
```

2. Открой проект в **Visual Studio**

3. Запусти:

```
F5
```

---

## 🎯 Управление

* ⬆️⬇️⬅️➡️ — движение змейки
* Esc — выход / пауза (если реализовано)
